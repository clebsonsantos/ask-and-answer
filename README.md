## Uso
- No terminal, clone o projeto:
```
https://github.com/clebsonsantos/ask-and-answer.git
```
- Entre na pasta do projeto projeto:
```
cd ask-and-answer
```
- Instale as dependências:
```
npm install
```

- Execute a aplicação
```
nodemon index.js
```
